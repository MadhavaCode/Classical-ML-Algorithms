# classical-ml-algorithms

13 notebooks covering classical supervised and unsupervised machine learning with real-world datasets.

## 📓 Notebooks

| Notebook | Algorithm | Dataset |
|---|---|---|
| `Simple_Multiple_Linear_Regression.ipynb` | OLS Linear Regression | TV Marketing, Advertising (MSE + R²) |
| `Logistics_Regression.ipynb` | Logistic Regression | Hearing test dataset |
| `Ridge_Lasso_Regularization.ipynb` | Ridge & Lasso | Regression regularization comparison |
| `KNN.ipynb` | K-Nearest Neighbours | Breast Cancer, Diabetes (k sweep 1–20) |
| `K-Means.ipynb` / `K_Means1.ipynb` | K-Means Clustering | Customer segmentation |
| `Agglomerative_Clustering.ipynb` | Hierarchical Clustering | Dendrogram analysis |
| `SVM_Support_Vector_Machine.ipynb` | Support Vector Machine | Binary classification |
| `Decision_Tree_and_Random_Forest.ipynb` | Decision Tree + Random Forest | Play Tennis (entropy), Iris |
| `Boosting.ipynb` | AdaBoost / Gradient Boosting | Ensemble methods |
| `NaiveBayes.ipynb` | Gaussian Naive Bayes | Probabilistic classification |

## 🛠 Tech Stack
`Python` · `scikit-learn` · `NumPy` · `Pandas` · `Matplotlib` · `Seaborn` · `mlxtend`
